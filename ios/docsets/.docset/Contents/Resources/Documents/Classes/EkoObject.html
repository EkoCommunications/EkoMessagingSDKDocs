EkoObject


@interface EkoObject <__covariant ObjectType> : NSObject

Proxy for an object

  • -observeWithBlock:

    Adds a block to observe changes in the proxy. Observers can be notified when the object is available. In case the object doesn’t exist or was unavailable an error will be passed.

    Declaration

    Objective-C

    - (nonnull EkoNotificationToken *)observeWithBlock:
    (nonnull void (^)(EkoObject<ObjectType> *_Nonnull,
                      NSError *_Nullable))block;

    Parameters

    block

    The block to execute when the request result has changed

    Return Value

    EkoNotificationToken object that should be retained to continue to observe.

  • -observeOnceWithBlock:

    Adds a block to observe a change in the proxy exactly once. Observers can be notified when the object is available. In case the object doesn’t exist or was unavailable an error will be passed.

    Parameters

    The block to execute exactly once when the request result has changed

    Return Value

    EkoNotificationToken object that should be retained to continue to observe.
