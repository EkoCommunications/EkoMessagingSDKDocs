EkoObject

@interface EkoObject <__covariant ObjectType>

Proxy for an object

  • -observeWithBlock:

    Adds a block to observe changes in the proxy. Observers can be notified when the object is available. In case the object doesn’t exist or was unavailable an error will be passed.

    Declaration

    Objective-C

    - (nonnull EkoNotificationToken *)observeWithBlock:
    (void (^_Nonnull)(ObjectType _Nullable, int *_Nullable))block;

    Swift

    func observe(_ block: @escaping (ObjectType?, UnsafeMutablePointer<Int32>?) -> Void) -> EkoNotificationToken

    Parameters

    block

    The block to execute when the request result has changed

    Return Value

    EkoNotificationToken object that should be retained to continue to observe.
