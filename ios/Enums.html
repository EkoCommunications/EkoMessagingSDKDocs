  • EkoErrorCode

    Errors that can happen within the SDK

    • UnauthorizedError: The action is not allowed with the current logged-in user
    • UserIsMuted: The action cannot be completed because the user is muted in the channel
    • ChannelIsMuted: The action cannot be completed because the entire channel is muted
    • UserIsBanned: The action cannot be completed because the user is banned
    • MaxRepetitionExceed: The action cannot be completed because it has been done too frequently
    • BanWordFound: The action cannot be completed because a blacklist word has been detected
    • GlobalBanError: The logged-in user is banned globally from using the SDK
    • InvalidParameter: The action cannot be completed because a parameter is invalid
    • ErrorQueryInProgress: There’s a query already in progress
